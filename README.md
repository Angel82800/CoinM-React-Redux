# CoinMetro Client

This project is built by React 15.4.2

## Getting Started

### Main Function

React, Redux, WebPack

### Prerequisites

```
NPM >=6.5.0
Node >= 3
```

### Installing

```
npm install
```

### Running the Tests

```
npm start
```

### Building

```
npm build
```
